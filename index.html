Timestamp API - FreeCodeCamp Project by Christopher McLean

Input

A string containing either a unix timestamp or a natural language date (example: January 1, 2016)

Output

A JSON object containing the "unix" and "natural" elements, corresponding to the date provided. The natural element will be in the format Month Date, Year.

On invalid input both properties will be null.

Usage examples

Input Output
https://timestamp-api-fcc-chris-mclean.herokuapp.com/1487462400 { "unix":1487462400, "natural":"February 19, 2017" }
https://timestamp-api-fcc-chris-mclean.herokuapp.com/Jan%201%202016 { "unix":1451606400, "natural":"January 1, 2016" }
https://timestamp-api-fcc-chris-mclean.herokuapp.com/November%2018,%202016 { "unix":1487462400, "natural":"November 18, 2016" }
